## About

CatiaMenuWin32 is a lightweight native Win32/C application that acts as a macro launcher
for PyCATIA scripts used with CATIA V5. It syncs scripts directly from GitHub repositories,
presents them as clickable buttons, and executes them via the local Python interpreter.
